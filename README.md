# 개요
본 프로젝트는 고전 게임 Pong을 채팅 기능과 함께 웹서비스로 구현한 프로젝트입니다.</br>
프론트엔드 프레임워크로는 Svelte를 사용했으며</br>
백엔드 프레임워크로는 NestJs를 사용하였고,</br>
DBMS로는 PostgreSQL을 사용하였습니다.
